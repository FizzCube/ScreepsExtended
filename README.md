# ScreepsExtended

A Chrome extension providing additional quality-of-life improvements for [Screeps](https://screeps.com/).

## Manual Installation (Chrome)

Follow these steps to load the extension in Chrome:

1. **Download or clone this repository**
   - Click **Code â†’ Download ZIP**, then extract it  
     **or**  
     - Clone it:
       ```bash
       git clone https://github.com/FizzCube/ScreepsExtended.git
       ```

2. **Open Chrome's Extensions page**
   - Go to: `chrome://extensions/`

3. **Enable Developer Mode**
   - Toggle on **Developer mode** in the top-right corner.

4. **Load the extension**
   - Click **Load unpacked**
   - Select the folder containing the extension's files (the root project directory)

5. **Verify installation**
   - The extension should now appear in your extensions list.
   - If needed, pin it to your toolbar for quick access.

## Updates

To update the extension, pull the latest changes (or re-download the ZIP) and reload the extension using the **Reload** button on the extensions page.
